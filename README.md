# 🍰 Dessert Shop Web App

A modern, conversion-focused dessert ordering platform built with pure HTML, CSS, and JavaScript — designed to deliver a smooth and intuitive user experience without relying on heavy frameworks.

## 🚀 Key Features

- 🛒 Advanced Cart System  
  Dynamic cart management with real-time updates and total calculation.

- 📦 Smart Order Summary  
  Automatically generates a clean and structured order summary.

- 📱 WhatsApp API Integration  
  One-click order submission directly to the business via WhatsApp with a pre-filled message.

- ⚡ Fast & Lightweight  
  Fully client-side application — no backend required.

- 💻 Responsive UI/UX  
  Optimized for both mobile and desktop users.

## 🧠 Technical Highlights

- Vanilla JavaScript (no frameworks)
- Dynamic DOM manipulation
- Event-driven architecture
- URL-encoded message generation for WhatsApp API
- Clean and maintainable code structure

## 🎯 Purpose

This project demonstrates how small businesses can launch a fully functional online ordering system quickly and efficiently — without complex infrastructure.

## 🌟 Why This Project Stands Out

Unlike traditional e-commerce platforms, this solution removes friction by allowing customers to place orders instantly through WhatsApp — a familiar and widely-used communication channel.
